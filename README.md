# A 3D Implementation of Fluid Simulation Using Laplacian Eigenfunctions in JAX

## Setup
```sh
$ uv venv -p /opt/homebrew/bin/python3.11
$ source .venv/bin/activate
```

## Install dependencies
```sh
$ uv pip install -e .[dev]
```

## Run the first scene
```sh
$ uv run cube.py
```

## Run the second scene
```sh
uv run cuboid.py
```
